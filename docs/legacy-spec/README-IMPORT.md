## Legacy spec — что это

Эти файлы — старая итерация спецификации SCC (импортированы из `Downloads/files (1)/` 2026-04-18).
Большая часть уже реализована или пройдена. Хранятся как архив для:
- ссылок из `docs/backlog.md` (раздел Stage C)
- быстрого ответа на «а как это было задумано раньше?»
- идей, которые можно достать в будущем (cannibalization detector, sale prep checklist и т.д.)

**Не используем как спецификацию текущего проекта.** Действующая спецификация — `CLAUDE.md` в корне репозитория и `docs/business-model.md`.

### Источники
- `CLAUDE.md`            — общая спека SCC старой версии
- `AGENTS.md`            — спека агентной системы (детальные пороги: CWV, keyword density, etc.)
- `ADMIN_AND_VALUATION.md` — административные функции и оценка сайтов (формулы, market benchmarks)
- `PROJECT_CONTEXT.md`   — краткий обзор
- `README.md`            — быстрый старт старой версии
- `seo-command-center.jsx` — single-file UI прототип (не использовать: прямой вызов Anthropic из браузера = security hole)
