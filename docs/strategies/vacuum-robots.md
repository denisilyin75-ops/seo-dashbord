## 1. Почему роботы-пылесосы — приоритет #2

- **Личная экспертиза владельца** — Dreame H12 Core в активном использовании, реальный опыт пользователя
- **Высокие чеки** (20-170k ₽) — премиум-модели (X50, L50, Matrix10) дают RPM выше среднего
- **Сезонность умеренная** — интерес круглый год с пиками март-апрель (весенняя уборка) и октябрь-декабрь (подарки)
- **Глубокая линейка Dreame** — в одной линейке десятки моделей с расшифровкой суффиксов (Plus/Ultra/Pro Ultra/Master/s) — идеально для глубокой экспертизы
- **Dreame vs Roborock** — постоянная битва, каждое сравнение даёт хороший трафик

## 2. Границы рубрики

**Входит:**
- Роботы-пылесосы (автономные)
- Вертикальные моющие пылесосы (Dreame H-серия и др.)
- Паровые швабры
- Пылесосы для лестниц/авто (если появятся логичные поводы)

**Не входит:** обычные пылесосы с мешком/без, центральные пылесосы (другая аудитория, чаще B2B).

## 3. Целевая аудитория и запросы

### Персоны

1. **Первая покупка робота** (бюджет 20-50k, дом без ковров)
   - Запросы: «какой робот-пылесос выбрать», «робот-пылесос для дома», «робот-пылесос с влажной уборкой до 30000»
   - ~40% трафика, высокая конверсия

2. **Апгрейд на премиум** (50-150k, есть старый, хочет новый с AI)
   - Запросы: «dreame x50 обзор», «roborock s8 vs dreame l20», «лучший робот-пылесос 2026»
   - ~35% трафика

3. **Узкие сценарии** (ковры, питомцы, высокие пороги)
   - Запросы: «робот-пылесос для ковров», «робот-пылесос для животных», «робот-пылесос с высоким проходом порогов»
   - ~15% трафика

4. **Энтузиасты/обновление** (следят за новинками)
   - Запросы: «новые роботы-пылесосы 2026», «dreame matrix10 обзор»
   - ~10% трафика

### Кластеры запросов

- **Транзакционные** (review): «обзор [МОДЕЛЬ]», «отзывы [МОДЕЛЬ]»
- **Сравнительные**: «[X] vs [Y]», «топ роботов-пылесосов», «dreame vs roborock»
- **Информационные**: «как выбрать», «робот-пылесос с [ФИЧА]», «какая модель для [СЦЕНАРИЙ]»
- **Проблемные**: «робот не едет», «застревает», «ошибка [КОД]», «как чистить»

## 12. E-E-A-T стратегия рубрики

### Экспертиза автора

- **Личный опыт Dreame H12 Core** — использовать как "якорь" экспертизы: «пишу как владелец Dreame H12 Core 2 года»
- Блок «Автор» в каждой статье: фото (если возможно), опыт, дата написания
- Ссылки на другие обзоры на сайте («я также тестировал Dreame L10s...»)

### Доверие

- Видимые даты обзоров
- Метка «обновлено» при refresh
- Ссылки на официальные сайты для проверки ТТХ
- Честные минусы в каждом обзоре

### Опыт

В каждом review-обзоре — блок «Реальный опыт» со словами, которые не из ТТХ:
- Уровень шума при уборке (дБ если замеряли)
- Как справляется с порогами реально (а не по заявлениям)
- Проскальзывает ли щётка на коврах
- Насколько часто застревает
- Сколько реально работает от одной зарядки (vs заявленное)
- Удобство приложения (какие функции работают на русском)

**Для моделей без личного тестирования** — честно пишем «оценка на основе ТТХ, видео-обзоров и отзывов владельцев» + ссылки на источники.

## 13. Ключевые темы для глубокой экспертизы

Для рубрики характерна высокая техническая сложность. Полезно заранее разобрать:

### Технологии навигации
- LiDAR (обычный / убирающийся)
- 3D структурированный свет
- RGB/AI-камеры
- LED-подсветка
- Комбинации

### Типы швабр
- Лоток (D-серия)
- Двойные вращающиеся
- Роликовая (Aqua10)
- Автоматическая смена типа (Matrix10)

### Автостанции
- Только зарядка
- Автоопустошение пылесборника
- Стирка швабр (холодная / тёплая / горячая до 100°C)
- Сушка швабр (тёплый / горячий воздух)
- Детектор грязи
- Подключение к водопроводу (Master-серия)

### Проходимость
- Стандарт ~20мм
- EasyLeap 39мм (L40s Ultra+)
- ProLeap 60мм (L50, X50)
- Экстра 88мм (X60)

Эти темы — бэкграунд для всех обзоров. Каждая модель описывается через них.
